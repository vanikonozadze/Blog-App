# პროექტის გაშვება

- npm install
- პროექტის გასაშვებად შეგვიძლია გამოვიყენოთ ბრძანება npm run start-project, ეს ქომანდი დასტარტავს DB-ს და გაუშვებს ng serve ქომანდს

# პროექტში გამოყენებულია

- Angular 19+
- NgRx state management
- RxJs
- Services
- Crud operations
- Routing
- Guards
- Resolvers
- Async pipes
- Reactive Forms
- Custom directives
- Signals
- Reusable components
- Unit tests
